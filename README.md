## :question:사용법
1. npm install과 npm start를 하시면 프로젝트를 구동할 수 있습니다.
2. 저희는 데이터베이스(백엔드)를 따로 구현하지 않았기 때문에, 세션&로컬 스토리지를 활용했습니다. 문제가 생긴다면, 로컬 스토리지를 전부 지우시고 새로고침을 하기 바랍니다.
